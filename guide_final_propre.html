1. Introduction

Bienvenue dans PLR Vet. Ce logiciel permet l'analyse objective du réflexe photomoteur (PLR) chez l'animal via une caméra infrarouge dédiée.

2. Gestion des Patients (Accueil)

Au lancement, l'écran d'accueil s'affiche. Il est divisé en trois zones principales pour une gestion fluide de votre patientèle.

Fig 1. Écran d'accueil : Liste des patients (gauche) et fiche détaillée (droite).

2.1 Base de Données (Gauche)

Cette liste contient tous vos patients enregistrés. Vous pouvez filtrer la liste en tapant un nom ou un numéro de puce dans la barre Rechercher....

  • Colonnes disponibles : Nom, Espèce, ID / Puce.

2.2 Création et Édition (Droite)

Pour ajouter un animal, cliquez sur le bouton bleu en bas à gauche : + CRÉER NOUVEAU PATIENT.

Remplissez ensuite la Fiche Patient à droite :

  • Champs obligatoires (*) : Nom, ID (Puce/Tatouage).
  • Détails : Espèce (Chien, Chat, NAC), Race, Sexe, Date de naissance.
  • Notes : Zone de texte libre pour vos remarques cliniques.

Cliquez sur 💾 Enregistrer pour valider.

2.3 Accès à l'Examen

Une fois le patient sélectionné ou créé, son historique s'affiche dans le panneau "Historique du Patient". Pour lancer une nouvelle session, cliquez sur le grand bouton vert : OUVRIR DOSSIER D'EXAMEN.

Fig 2. Interface d'examen en temps réel avec retour vidéo et contrôles.

Cette fenêtre est votre outil de travail principal. Elle permet de contrôler la caméra, stimuler l'œil et visualiser les résultats.

  • Historique (Bas) : Tableau récapitulatif des examens de la session (Heure, Oeil, Couleur Stim, Type).

4. Réaliser un Examen (Pas à Pas)

Étape 1 : Préparation

Positionnez la caméra devant l'œil de l'animal. Ajustez la distance jusqu'à ce que l'image soit nette. Le logiciel doit détecter la pupille (cercle de suivi).

Étape 2 : Lancement

  1. Vérifiez que l'œil sélectionné (OD/OG) est correct.
  2. Choisissez la couleur du stimulus (par défaut Bleu).
  3. Cliquez sur ▶ LANCER EXAMEN.

Étape 3 : Séquence Automatique

Le protocole standard (configurable) s'exécute :

  • 0s - 1s : Baseline (Mesure au repos).
  • 1s - 2s : FLASH (Stimulation lumineuse). Une bande jaune apparaît sur le graphique.
  • 2s - 8s : Réponse (Suivi de la constriction et redilatation).
Fig 3. Analyse post-examen : La zone jaune indique le flash, la courbe bleue la réponse pupillaire.

La courbe bleue montre la dynamique pupillaire. Vous pouvez survoler la courbe pour voir les valeurs précises (Temps / Diamètre).

5. Configuration et Paramètres

Accessible via le menu Options > Paramètres, cette fenêtre permet de personnaliser le logiciel.

5.1 Onglet Protocole

Fig 4. Configuration des durées et du type de flash.

Définit la chronologie de l'examen. Les valeurs par défaut sont :

  • Baseline : 1,00 s
  • Flash : 1,00 s (Durée de la stimulation)
  • Durée TOTALE : 8,00 s
  • Couleur par défaut : Bleu (480nm)

5.2 Onglet Détection

Permet de calibrer l'algorithme de vision :

  • Seuil (Threshold) : Valeur de gris (0-255) pour isoler la pupille. Par défaut à 147.
  • ROI (Region of Interest) : Taille de la zone d'analyse (400x200 par défaut) pour accélérer le traitement.

5.3 Onglet Macros

Gérez ici vos phrases types pour les rapports. Tapez votre texte, cliquez sur Ajouter. Elles seront ensuite disponibles dans la liste déroulante de la fenêtre d'examen.

5.4 Onglet Clinique

Renseignez les coordonnées de votre clinique (Nom, Adresse, Praticien) et importez votre Logo. Ces informations apparaîtront sur les rapports PDF exportés.

6. Dépannage

L'image est trop sombre ou trop claire.
Ajustez le curseur Seuil dans la fenêtre d'examen. Si le seuil est trop bas, tout est noir. S'il est trop haut, le logiciel détecte des ombres comme étant la pupille.

Le graphique reste plat.
Vérifiez que la case "Voir Bruit (Brut)" est décochée pour voir les données filtrées. Assurez-vous que le cercle vert suivait bien la pupille pendant le flash.
